Website Accessibility & Usability

What I ensured was widely accessible and usable.

  • Images
  • Straightforward layout
  • Text
  • Screen size
  • Document structure

Images

I used alt text with any images such as pictures or videos so that if there was a reason that the image could not be shown, or if the user is using a text-only setting so that they will still be able to understand the sites content.

Straightforward layout

While designing the website I knew that I wanted it to be navigated easily and be visually appealing, therefore a two column format with a header and fixed menu were used.

Text

To ensure readability and proffesionalism, I chose the Arial font and large text size

Screen size

To make sure that the website is available to be viewed on different devices and window sizes I had the two segments stack on top of eachother if the size of the website went below a ceratin amount.

Document structure

When users navigate to different pages within the website I wanted to make sure that the layout remained consistent and clear. And that users understood where on the website they were able to navigate to.
